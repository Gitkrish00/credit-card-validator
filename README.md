## Credit Card Validator

A basic Python GUI application that checks whether a credit card number is valid using the Luhn algorithm.

## Features
  Simple graphical interface using Tkinter
  Validates 16-digit credit card numbers
  Accepts input with spaces or dashes
  Displays validation result using popup messages

The program:

Takes user input
Removes spaces and dashes
Checks if the number is 16 digits
Uses the Luhn algorithm to validate the number
How to Run
Make sure Python is installed
Save the file as credit_card_validator.py
Run the program:
python credit_card_validator.py
